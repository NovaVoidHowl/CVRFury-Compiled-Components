# VRChat Avatar SDK3 Stub

A very lightweight DLL set specifically configured for CVRFury to support extraction of VRC SDK Components data,
for the Avatar conversion Feature

## Warnings
- VRC SDK must not be installed in the project

## Supported classes

### Avatar Components (VRCAVstub)
- VRCAvatarDescriptor
- VRCSpatialAudioSource
- VRCHeadChop
- VRCAvatarParameterDriver

### Avatar ScriptableObjects (VRCAVstub)
- VRCExpressionParameters
- VRCExpressionsMenu

### PhysBones Components (VRCPBstub)
- VRCPhysBone
- VRCPhysBoneCollider

### Contact Components (VRCPCstub)
- VRCContactReceiver
- VRCContactSender

### Constraint Components (VRCPConStub)
- VRCAimConstraint
- VRCLookAtConstraint
- VRCParentConstraint
- VRCPositionConstraint
- VRCRotationConstraint
- VRCScaleConstraint

### Pipeline Management (VRCPMstub)
- PipelineManager
